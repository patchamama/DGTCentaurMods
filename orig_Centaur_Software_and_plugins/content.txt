Change the password of the image saved in the sdcard:
 1. `lsblk -f` OR `sudo fdisk -l` to see the partitions.
 2. Mount partition: `sudo mkdir -p /mnt/rpi && sudo mount /dev/sdX2 /mnt/rpi` (change /dev/sdX2 for the partition with the ext4 and content: /dev /home...)
 3. Create password to be replaced, example: raspberry: `openssl passwd -1 raspberry`. Replace in the `/etc/shadow` the password of pi or root for the new password. (`nano /etc/shadow`) 
 4. `sudo umount /dev/sdX2`
